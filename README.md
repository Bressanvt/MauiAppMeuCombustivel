# MauiAppMeuCombustivel

App pra saber se compensa abastecer com etanol ou gasolina. A regra e a classica: se o etanol custar ate 70% do valor da gasolina, ele compensa mais. Os calculos sao salvos no banco pra consultar depois.

**Conceitos novos:** injecao de dependencia com AddSingleton, SQLite com inicializacao lazy no DatabaseService, model com PrimaryKey e AutoIncrement pra historico de calculos.
